# P03 · 拍照录入 Capture（`02_capture.html`）

## 1. 状态枚举
- `default` · 相机预览 · 底部三按钮：相机 / 相册 / 手动
- `selecting` · 已选图 · 预览 + 裁剪
- `compressing` · 客户端压缩（>2MB 触发）· 进度条
- `uploading` · PUT OSS · 进度条 `progress`
- `error` · 网络失败 / OSS 超时 → 重试按钮
- `size_over` · 图 > 10MB → 吐司 "单张 ≤ 10MB"（SC-11.AC-1 error_paths.1）
- `mime_invalid` · 非白名单 mime → 吐司 "仅支持 JPG/PNG/HEIC"

## 2. 数据字段 → API 映射
| UI 字段 | API |
|---|---|
| 原图 file | `POST /files/presign` → `PUT {uploadUrl}` · SC-11.AC-1 |
| OCR 识别文本 | `POST /ocr/recognize`（百度 OCR 或 cloud）|
| 提交错题 | `POST /wrongbook/items {image_ref: fileKey, ocr_text}` |

## 5. 响应式
- 全屏 · 底部 SafeArea · 拍照预览 4:3 固定比例
- 小程序 `camera` 组件 · frame 1080×1440

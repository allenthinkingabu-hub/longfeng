# P11 · 日历月视图 Calendar Month（`10_calendar_month.html`）

## 1. 状态枚举
- `default` · 当前月 · 每日格子显示 due 数量 + 已完成 check
- `loading` · 月聚合查询中
- `navigating` · 左右切月 · prev/next slider
- `empty` · 本月无任何计划 · 显示 "本月暂无复习任务"

## 2. 数据字段 → API 映射（SC-10 日历节点绑定）
| UI 字段 | API |
|---|---|
| 月聚合 | `GET /review-plans?date_range=YYYY-MM` + `GET /calendar/nodes?month=YYYY-MM` · SC-10.AC-1 |
| 单日详情 | 点击 day cell → 跳 P12 event_detail |

## 3. testid
| 元素 | testid |
|---|---|
| 月标题 | `calendarmonth.title.text` |
| 切月 prev | `calendarmonth.prev.btn` |
| 切月 next | `calendarmonth.next.btn` |
| day cell | `calendarmonth.day.cell-{YYYY-MM-DD}` |
| day dot (due) | `calendarmonth.day.dot-{YYYY-MM-DD}` |

## 4. 埋点
- `calendarmonth_page_view` · `{month}`
- `calendarmonth_day_click` · `{date, due_count}`
- `calendarmonth_nav` · `{direction: prev|next}`

## 5. 响应式
- 7x5 grid · 每 cell 48×48 (mobile) / 64×64 (tablet)

## 6. 深色模式
- cell bg `--tkn-color-surface-dark-1` · due dot `--tkn-color-accent-primary`

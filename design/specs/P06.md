# P06 · 错题本列表 Wrongbook List（`05_wrongbook_list.html`）

## 1. 状态枚举
- `default` · 列表展示 · 按创建时间倒序 · 20/页
- `empty` · 用户无错题 · 引导拍照
- `loading` · 骨架屏 · 20 条占位
- `filtering` · 筛选器展开 · 科目/标签/日期
- `deleting` · 软删 loading（SC-04 错题软删除）

## 2. 数据字段 → API 映射
| UI 字段 | API |
|---|---|
| 列表 | `GET /wrongbook/items?page=1&size=20&subject=math` |
| 软删 | `DELETE /wrongbook/items/{id}`（返 204 · `deleted_at=now()`）|
| 恢复 | `POST /wrongbook/items/{id}/restore` |
| 筛选 | query param `subject` / `tag` / `date_range` |

## 3. testid
| 元素 | testid |
|---|---|
| 列表 item | `wronglist.item.card-{id}` |
| 筛选器入口 | `wronglist.filter.btn` |
| 科目筛选 chip | `wronglist.filter.subject-{math|physics|chinese|english}` |
| 删除 swipe action | `wronglist.item.delete-{id}` |
| 恢复 toast action | `wronglist.restore.btn` |

## 4. 埋点
- `wronglist_page_view` · `{total_count, filter_subject}`
- `wronglist_item_click` · `{item_id}`
- `wronglist_delete` · `{item_id}`
- `wronglist_restore` · `{item_id}`

## 5. 响应式
- H5 · 1-col on mobile · 2-col on tablet (≥ 768px)
- 小程序 · `scroll-view` + lower bound trigger

## 6. 深色模式
- list bg `--tkn-color-bg-dark` · card `--tkn-color-surface-dark-1`

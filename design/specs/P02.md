# P02 · 首页 Home（`01_home.html`）

## 1. 状态枚举
- `default` · 用户登录后 · 今日无到期复习 → 显示空态 "今日无复习任务" + 推荐拍照
- `with_due` · 今日 due 节点 ≥ 1 → 顶部卡显示数量 + TOP3 薄弱科目
- `loading` · 启动加载中 · 骨架屏 · `<Skeleton>` 组件
- `error` · API 失败 · 顶部黄色 Banner + 下拉重试
- `empty` · 新用户 · 无任何错题 → 引导拍照录入

## 2. 数据字段 → API 映射
| UI 字段 | API |
|---|---|
| 今日复习数 | `GET /review-plans?date=today` body.data.length |
| TOP3 薄弱 | `GET /review-stats?range=week` body.data.topWeak[0..2] |
| 近期错题数 | `GET /wrongbook/items?range=7d` body.count |

## 3. testid
| 元素 | testid |
|---|---|
| 拍照入口 FAB | `home.fab.capture` |
| 复习卡 | `home.today.review-card` |
| TOP3 卡 | `home.insight.top3-card` |
| 底部 TabBar | `home.tabbar.tab-{home,wrong,calendar,me}` |

## 4. 埋点
- `home_page_view` · 进入首页 · `{has_due: bool, total_wrong: int}`
- `home_capture_click` · 点 FAB 拍照
- `home_review_click` · 点今日复习卡 · `{due_count}`

## 5. 响应式
- iPhone notch 顶部安全区 `env(safe-area-inset-top)`
- 小程序 `navigation-bar` 扩展到通知栏

## 6. 深色模式
- 卡片 bg `--tkn-color-surface-dark-1` · text-primary `--tkn-color-text-on-dark`

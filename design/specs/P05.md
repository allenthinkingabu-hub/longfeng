# P05 · AI 解析结果 Result（`04_result.html`）

## 1. 状态枚举
- `success` · 完整解析 · 显示：题干 + 标签（subject/grade/knowledge_points JSONB）+ AI 讲解 + 相似题
- `partial` · 部分字段缺 · 显示已有 + 灰底占位
- `degraded` · SC-06.AC-1 失败降级 · 占位 + "点击重试" CTA
- `saving` · 用户点"加入错题本" · CTA loading
- `saved` · 已入库 · CTA 变"已加入" + 灰底

## 2. 数据字段 → API 映射
| UI 字段 | API |
|---|---|
| 题干 stemText | `GET /ai/analysis/{itemId}.data.stem` |
| 标签 | `.data.knowledge_points[]` |
| AI 讲解 | `.data.solution_steps` |
| 相似题 Top3 | `GET /ai/similar?item_id=X` |
| 加入错题本 | `POST /wrongbook/items {ocr_text, ai_result_id}` |
| 重试 AI | `POST /ai/retry/{itemId}` |

## 3. testid
| 元素 | testid |
|---|---|
| 题干 | `result.stem.text` |
| 标签 Chip | `result.tag.chip-{subject}` |
| AI 讲解 | `result.solution.text` |
| 相似题 Card | `result.similar.card-{n}` |
| 加入错题本 CTA | `result.save.btn` |
| 重试 AI CTA | `result.retry.btn` |

## 4. 埋点
- `result_page_view` · `{item_id, status}`
- `result_save_click` · `{item_id}`
- `result_similar_click` · `{similar_item_id}`
- `result_retry_click` · `{item_id, attempt_n}`

- H5 · 375-1023 全屏 · 标签横向滚 · 题干最大宽 640
- 小程序 · `scroll-view` 分区

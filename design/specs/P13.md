# P13 · 通知 Notifications（`12_notifications.html`）

## 1. 状态枚举
- `default` · 通知列表 · 复习提醒 + 掌握庆祝 + 家长提醒
- `empty` · 无通知 · 显示 "一切静好"
- `unread_badge` · 顶部 TabBar icon 显示红点
- `marking_read` · 批量标已读 loading
- `subscribe_prompt` · 首次未授权微信订阅消息 → 显示授权引导（骨架 · S8 接）

## 2. 数据字段 → API 映射
| UI 字段 | API |
|---|---|
| 通知列表 | `GET /notifications?page=1&size=20` |
| 标已读 | `PUT /notifications/{id}/read` |
| 全部已读 | `PUT /notifications/mark-all-read` |
| 订阅授权 | 微信 `requestSubscribeMessage` API（S8 接）|

## 3. testid
| 元素 | testid |
|---|---|
| 通知 item | `notifications.item.card-{id}` |
| 已读标记 | `notifications.item.read-{id}` |
| 全部已读 CTA | `notifications.markall.btn` |
| 授权引导 | `notifications.subscribe.prompt` |

## 4. 埋点
- `notifications_page_view` · `{unread_count}`
- `notifications_item_click` · `{notif_id, type: review_due|mastered|parent_view}`
- `notifications_subscribe_grant` · `{granted: bool}`

## 5. 响应式
- 单列 · 每 item 高 72

## 6. 深色模式
- 未读 bg `--tkn-color-surface-dark-3` · 已读 `--tkn-color-surface-dark-1`

# P08 · 今日复习 Review Today（`07_review_today.html`）

## 1. 状态枚举
- `has_due` · 今日 due 节点 ≥ 1 · 卡片队列 + "开始复习" CTA
- `empty` · 今日无 due · 显示 "所有任务已完成 · 明天再来"
- `loading` · 复习计划加载中
- `error` · API 失败

## 2. 数据字段 → API 映射（SC-07 消费）
| UI 字段 | API |
|---|---|
| 今日 due 列表 | `GET /review-plans?date=today` · S5 owner |
| 按 SC-07.AC-1 · 每条显示 node_index 0..6 · next_review_at | - |
| 开始复习 CTA | 导航 P09 review_exec |

## 3. testid
| 元素 | testid |
|---|---|
| due 卡片 | `reviewtoday.due.card-{plan_id}` |
| 开始复习 CTA | `reviewtoday.start.btn` |

| 进度条 | `reviewtoday.progress.bar` |

# P17 · 分享卡 Shared（`16_shared.html`）

## 1. 状态枚举
- `valid` · 分享链接有效（TTL 7d）· 显示错题卡 + 水印
- `expired` · TTL 过期 · 显示 "分享已失效"
- `not_found` · share_token 非法 · 404
- `viewer_anon` · 未登录访问 · 底部"注册查看更多"引导
- `viewer_logged` · 已登录访问 · 显示"加入我的错题本" CTA

## 2. 数据字段 → API 映射
| UI 字段 | API |
|---|---|
| 分享内容 | `GET /share/{token}` → `{wrong_item, watermark_text, expires_at}` · TTL 7d |
| 访问日志 | 自动 POST（服务端记 share_access_log）|
| 加入我的 | `POST /wrongbook/items/adopt {source_share_token}` |

## 3. testid
| 元素 | testid |
|---|---|
| 分享题干 | `shared.wrong.stem` |
| 水印 | `shared.watermark.text` |
| 加入我的 CTA | `shared.adopt.btn` |
| 注册引导 | `shared.register.cta` |

## 4. 埋点
- `shared_page_view` · `{share_token, viewer_type: anon|logged}`
- `shared_adopt_click`
- `shared_register_click`

## 5. 响应式
- 居中 · 最大宽 640 · 水印覆盖全图片

## 6. 深色模式
- 水印半透 `--tkn-color-text-on-dark`

# P04 · AI 解析中 Analyzing（`03_analyzing.html`）

## 1. 状态枚举
- `queued` · wrongbook.item.analyzed 事件未到 · 显示 loading spinner + 文案 "AI 分析中 · 预计 3s"
- `in_progress` · S4 处理中 · 脉冲动画
- `completed` · 3s 内成功 → 自动跳 P05 Result
- `timeout` · 3s+ 未收到事件 · 显示 "耗时较久 · 继续等待 or 先去别处"
- `failed_fallback` · SC-06.AC-1 降级 · 占位卡 "AI 服务忙，仅存图片"（不阻塞主链路）

## 2. 数据字段 → API 映射
| UI 字段 | API |
|---|---|
| 轮询 AI 状态 | `GET /ai/analysis/{itemId}` body.data.status（completed/degraded/timeout）· SC-05.AC-1 · SC-06.AC-1 |
| 降级占位 | `{subject: "unknown", stemText: "[降级]"}` |

## 3. testid
| 元素 | testid |

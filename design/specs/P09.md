# P09 · 复习执行 Review Exec（`08_review_exec.html`）

## 1. 状态枚举
- `think` · 显示题干 · "想一想" · 底部"翻开答案" CTA
- `reveal` · 翻开 · 显示 AI 解析 + 4 档自评按钮（quality 0-5 映射）
- `submitting` · 点评分 · POST complete loading
- `success` · 200 返回 · 短暂 toast "已记录" + 自动下一题
- `conflict` · 409 乐观锁冲突 · toast "已由其他设备完成"
- `mastered` · 连续 3 次 ease≥2.8 · celebration 动画 + 跳 P10 done

## 2. 数据字段 → API 映射（SC-08.AC-1 消费）
| UI 字段 | API |
|---|---|
| 题干 | 从 P08 传入 plan_id · GET `/review-plans/{id}` |
| 提交评分 | `POST /review-plans/{id}/complete {quality}` · S5 owner |
| mastered 检测 | response.data.mastered=true · 显示 celebration |

## 3. testid
| 元素 | testid |
|---|---|
| 题干 | `reviewexec.stem.text` |
| 翻开答案 CTA | `reviewexec.reveal.btn` |
| 评分 "忘了" | `reviewexec.rate.forgot` (quality=0..2) |
| 评分 "模糊" | `reviewexec.rate.fuzzy` (quality=3) |
| 评分 "记得" | `reviewexec.rate.recall` (quality=4) |
| 评分 "掌握" | `reviewexec.rate.mastered` (quality=5) |
| Celebration 动画 | `reviewexec.celebration.anim` |

## 4. 埋点
- `reviewexec_reveal` · `{plan_id, think_duration_ms}`
- `reviewexec_rate` · `{plan_id, quality, ease_before, ease_after}`
- `reviewexec_mastered` · `{wrong_item_id, plan_id}`
- `reviewexec_conflict_409` · `{plan_id}`（乐观锁冲突率统计）

## 5. 响应式
- 单卡居中 · 最大宽 640 · 按钮 44×44pt（a11y）

## 6. 深色模式
- 卡片 `--tkn-color-surface-dark-1` · 按钮 variant 自适应

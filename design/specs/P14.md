# P14 · 设置 Settings（`13_settings.html`）

## 1. 状态枚举
- `default` · 设置列表（账号 / 通知 / 隐私 / 关于）
- `editing` · 某项编辑中 · 局部 loading
- `logout_confirm` · 退出确认 dialog
- `dark_toggle` · 暗色模式开关（前端持久化 localStorage）

## 2. 数据字段 → API 映射
| UI 字段 | API |
|---|---|
| 账号信息 | `GET /users/me` |
| 通知偏好 | `GET /users/me/notification-prefs` · `PUT` |
| 隐私设置 | `GET /users/me/privacy` · `PUT` |
| 退出 | `POST /auth/logout` |
| 账号注销 | `DELETE /users/me`（SC-11 A7 GDPR 软删 + 30d 硬删）|

## 3. testid
| 元素 | testid |
|---|---|
| 账号 item | `settings.account.row` |
| 通知偏好 switch | `settings.notif-prefs.switch-{review_due|mastered|parent}` |
| 暗色开关 | `settings.dark-mode.switch` |
| 退出 CTA | `settings.logout.btn` |
| 注销账号 link | `settings.delete-account.link` |

## 4. 埋点
- `settings_page_view`
- `settings_dark_mode_toggle` · `{enabled}`
- `settings_notif_prefs_change` · `{key, value}`
- `settings_logout`
- `settings_delete_account`

## 5. 响应式
- 单列 · 每 row 高 56

## 6. 深色模式
- 本页本身即开关源头 · 值持久化 `localStorage.theme`

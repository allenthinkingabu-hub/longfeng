# P01 · 登录 / 注册入口（`00_login.html`）

> 落地计划 §16.2 Sd.3 · 高保真页面规约 · 6 小节

## 1. 状态枚举

| 状态 | 描述 | 视觉差异 |
|---|---|---|
| `default` | 未输入手机号 · CTA 禁用 | CTA 灰底 `--tkn-color-button-default-light` |
| `typing` | 手机号输入中 · 格式校验进行时 | 输入框聚焦 · CTA 按规则激活 |
| `loading` | 点击"发送验证码"等待响应 | CTA loading spinner |
| `error` | 手机号格式非法 / 限流 429 | 红色文案 `--tkn-color-danger-text` 显示错误 |
| `success` | 验证码发出 · 倒计时 60s | CTA 变灰 + "60s" |
| `disabled` | 倒计时未到 · 二次点击 | CTA 全禁 |

## 2. 数据字段 → API 映射

| UI 字段 | API endpoint | payload / 返回 |
|---|---|---|
| 手机号 input | `POST /auth/sms-code` | `{phone: string}` → `{ttl_seconds: 300}` |
| 验证码 input | `POST /auth/verify` | `{phone, code}` → `{access_token, user_id}` |
| "匿名体验" 链接 | 导航至 P15 landing | - |

## 3. 可交互元素 → testid

| 元素 | testid |
|---|---|
| 手机号 input | `login.phone.input` |
| 验证码 input | `login.code.input` |
| 发送验证码 button | `login.send-code.btn` |
| 提交登录 button | `login.submit.btn` |
| 匿名体验 link | `login.guest.link` |

## 4. 埋点事件

| 事件名 | 触发时机 | Payload |
|---|---|---|
| `login_page_view` | 进入页面 | `{source: deeplink\|direct}` |
| `login_code_sent` | 验证码成功下发 | `{phone_hash}` |
| `login_success` | 验证码验证通过 | `{user_id, method: sms}` |
| `login_error` | 任一错误响应 | `{error_code, phone_hash}` |

## 5. 响应式断点

- ≥ 1024px（H5）：居中 480×640 卡片
- 375-1023px（H5 mobile）：全屏 · 顶部 logo 固定
- 小程序：`rpx` 单位 · 750rpx 基准 · 自动适配各机型

## 6. 深色模式预留

- 背景 `--tkn-color-bg-light` / `bg-dark` 自动切换
- CTA 蓝 `--tkn-color-accent-primary` 保持（iOS system blue 暗色同）
- 暂用 `prefers-color-scheme` · 本 Phase 不做人工切换（留 P14 Settings）

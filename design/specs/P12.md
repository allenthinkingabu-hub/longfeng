# P12 · 事件详情 Event Detail（`11_event_detail.html`）

## 1. 状态枚举
- `default` · 单日视图 · 列出 due plans + 已完成 + 未复习 · 时间线
- `completing` · 某项正在提交 · loading
- `no_due` · 选中某日无 due · 显示"当日无复习任务"
- `past_date` · 历史日期 · 仅显示已完成事件 · 无新动作

## 2. 数据字段 → API 映射
| UI 字段 | API |
|---|---|
| 当日 plan 列表 | `GET /review-plans?date=YYYY-MM-DD` |
| 复习历史 | `GET /review-events?date=YYYY-MM-DD` |
| 自定义事件 | `GET /calendar/nodes?date=YYYY-MM-DD`（SC-10）|

## 3. testid
| 元素 | testid |
|---|---|
| 时间线 item | `eventdetail.timeline.item-{plan_id}` |
| 完成状态标记 | `eventdetail.status.done-{plan_id}` |
| 去复习 CTA | `eventdetail.start-review.btn` |

## 4. 埋点
- `eventdetail_page_view` · `{date, due_count, done_count}`
- `eventdetail_start_click` · `{plan_id}`

## 5. 响应式
- 时间线竖向 · 每项高 80

## 6. 深色模式
- 时间线线条 `--tkn-color-text-tertiary` · item `--tkn-color-surface-dark-1`

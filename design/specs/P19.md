# P19 · 家长监督视图 Observer（`18_observer.html`）

## 1. 状态枚举
- `authorized` · 家长已加入家庭组 · 查看孩子学情
- `invite_pending` · 邀请未接受 · 显示邀请 CTA
- `no_child` · 无绑定孩子 · 引导邀请
- `child_private` · 孩子开启隐私模式 · 仅显示聚合 · 不显示具体错题

## 2. 数据字段 → API 映射（SC-14 家长监督）
| UI 字段 | API |
|---|---|
| 家庭组查询 | `GET /family/members`（只读聚合）|
| 孩子学情聚合 | `GET /review-stats?user_id={child}&range=week` · SC-09.AC-1（S5 提供数据源）|
| 错题清单（非隐私）| `GET /wrongbook/items?user_id={child}&aggregate=true` |
| 邀请 | `POST /family/invite {phone}` |

## 3. testid
| 元素 | testid |
|---|---|
| 孩子名字 | `observer.child.name-{child_id}` |
| 本周学情卡 | `observer.insight.week-card` |
| Top3 薄弱 | `observer.insight.top3` |
| 邀请 CTA | `observer.invite.btn` |
| 隐私告知 | `observer.privacy.notice` |

- `observer_page_view` · `{parent_id, child_count}`

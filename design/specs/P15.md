# P15 · Landing / 匿名引导（`14_landing.html`）

## 1. 状态枚举
- `default` · 未登录 · 显示产品介绍 + 3 道样题 + "立即体验" CTA
- `trying_sample` · 点样题 · 展开样题预览
- `cta_trial` · 点"立即体验" · 跳 P16 guest_capture
- `login_flow` · 点"已有账号" · 跳 P01 login

## 2. 数据字段 → API 映射
| UI 字段 | API |
|---|---|
| 3 道样题 | `design/seed/sample-questions.json`（Sd.8 产出 · 本地 asset）|
| 样题预览 | - |
| 立即体验 | 创建 anonymous session · `POST /anon/session {device_fp}` · SC-12 |

## 3. testid
| 元素 | testid |
|---|---|
| hero title | `landing.hero.title` |
| 3 样题 card | `landing.sample.card-{1|2|3}` |
| 立即体验 CTA | `landing.hero.cta.try-once` |
| 已有账号 link | `landing.login.link` |

## 4. 埋点
- `landing_page_view` · `{utm_source, utm_campaign}`
- `landing_sample_click` · `{sample_index}`
- `landing_try_click`
- `landing_login_click`

## 5. 响应式
- Desktop hero 1440×400 · mobile 全屏纵向

## 6. 深色模式
- hero 渐变 `--tkn-color-accent-primary` → `--tkn-color-accent-secondary`

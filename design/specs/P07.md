# P07 · 错题详情 Wrongbook Detail（`06_wrongbook_detail.html`）

## 1. 状态枚举
- `default` · 全量字段 · 图片 + 题干 + AI 解析 + 标签 + 复习历史
- `editing_tags` · 标签管理 sheet 弹出（SC-03）
- `loading_history` · 复习历史 loading
- `error` · 详情 API 失败 · 重试

## 2. 数据字段 → API 映射
| UI 字段 | API |
|---|---|
| 详情 | `GET /wrongbook/items/{id}` |
| 图片 | `GET /files/download/{fileKey}?variant=medium` · SC-11.AC-3 |
| 复习历史 | `GET /review-plans?wrong_item_id={id}` |
| 编辑标签 | `PUT /wrongbook/items/{id}/tags {tags: []}` |
| 删除 | `DELETE /wrongbook/items/{id}` |

## 3. testid
| 元素 | testid |
|---|---|
| 图片 | `wrongdetail.image.img` |
| 题干 | `wrongdetail.stem.text` |
| 标签 chip | `wrongdetail.tag.chip-{i}` |
| 编辑标签 CTA | `wrongdetail.tag-edit.btn` |
| 复习历史 item | `wrongdetail.history.item-{i}` |
| 删除 CTA | `wrongdetail.delete.btn` |

## 4. 埋点
- `wrongdetail_page_view` · `{item_id, from: list|deeplink}`
- `wrongdetail_tag_edit` · `{item_id, tags_before, tags_after}`
- `wrongdetail_delete` · `{item_id}`

## 5. 响应式
- 图片最大宽 1024px · 剪切安全区
- 小程序 · `image` 模式 `aspectFit`

## 6. 深色模式
- 图片 bg 黑 · 标签 `--tkn-color-surface-dark-2`

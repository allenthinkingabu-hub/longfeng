# P10 · 复习完成 Review Done（`09_review_done.html`）

## 1. 状态枚举
- `summary` · 本次复习完成 · 统计卡：n 题 · 正确率 % · ease 变化
- `celebration` · 全对 · 额外 confetti 动画
- `streak_extended` · 连续 n 天打卡 · 显示徽章
- `empty` · 直接跳转到 · 显示"暂无数据"（异常态）

## 2. 数据字段 → API 映射
| UI 字段 | API |
|---|---|
| 本次统计 | 本地 session state（P09 累积）|
| 历史趋势（可选） | `GET /review-stats?range=week` · SC-09.AC-1 |
| 打卡状态 | `GET /user/streak` |

## 3. testid
| 元素 | testid |
|---|---|
| 统计卡 | `reviewdone.summary.card` |
| 正确率数字 | `reviewdone.correct-rate.value` |
| 分享按钮 | `reviewdone.share.btn` |
| 回首页 CTA | `reviewdone.home.btn` |
| 再练一组 link | `reviewdone.again.link` |

## 4. 埋点
- `reviewdone_page_view` · `{count, correct_rate, duration_s}`
- `reviewdone_share_click`
- `reviewdone_again_click`

## 5. 响应式
- 居中 · 卡片最大宽 480

## 6. 深色模式
- 统计卡 `--tkn-color-surface-dark-2` · confetti 保持彩色

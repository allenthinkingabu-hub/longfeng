# P18 · 欢迎回归 Welcome Back（`17_welcomeback.html`）

## 1. 状态枚举
- `returning_user` · 未活跃 ≥ 14d 回归 · hero + 最近错题数 + 推荐复习
- `new_content` · 期间有新艾宾浩斯节点 due · 显示 "n 题等你复习"
- `streak_lost` · 连续打卡断了 · 温和提示 · 不扣分
- `loading` · 用户统计加载

## 2. 数据字段 → API 映射
| UI 字段 | API |
|---|---|
| 用户状态 | `GET /users/me/returning-summary` · `{days_away, due_count, streak_status}` |
| 推荐复习 | `GET /review-plans?date=today&recommended=true` |
| 本周学情 | `GET /review-stats?range=week` · SC-09.AC-1 |

## 3. testid
| 元素 | testid |
|---|---|
| hero | `welcomeback.hero.text` |
| 天数 counter | `welcomeback.days-away.value` |
| 推荐复习 CTA | `welcomeback.start-review.btn` |
| 查看学情 link | `welcomeback.insight.link` |

## 4. 埋点
- `welcomeback_page_view` · `{days_away, due_count}`
- `welcomeback_start_click`
- `welcomeback_dismiss` · 用户点×

## 5. 响应式
- 居中卡片 480×640

## 6. 深色模式
- hero 渐变柔和暖色 · 不突兀
